Schema for User model
1. Name
2. Email
3. Password
4. Avatar
5. Date

order of actions for registering new User
1. receive post request
2. extract new user information from body of request
3. Make sure no empty field was sent.
4. make sure email is in correct format.
5. make sure password and confirm-password match each other.
// 3,4 and 5 are all handled by one function 
6. make sure email is not already registered with another user.
7. if all these criteria are met, add information to database.
